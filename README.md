# Gaia-GIC-1-IsochroneSEDFit

The following Jupyter notebook and files utilize the Modules for Experiments in Stellar Astrophysics (MESA) Isochrones & Stellar Tracks to fit the broadband photometric data of Gaia-GIC-1 using PyMultiNest.
 
Table of Contents
* `Gaia-GIC-1-IsochronesSED Fitting.ipynb` - is the Jupyter notebook routine of performing the PyMultiNest fit using Isochrones, including a corner plot of the posterior distributions of the stellar parameters. 
*  `Chains/` contains the outputs from PyMultiNest fitting.
